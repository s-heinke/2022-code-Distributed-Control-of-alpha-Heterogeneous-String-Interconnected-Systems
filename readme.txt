This folder contains the files to reproduce the results presented in

S. Heinke and H. Werner, "Distributed Control of Î±-Heterogeneous String
Interconnected Systems".

The performance comparison and time domain simulations are contained in 'main.m'.
The comparison of the computational complexity is contained in 'main_computational.m'.

In order to run the code, the following Matlab packages are required:
- yalmip (https://yalmip.github.io/download/)

The code has been tested using:
- Matlab R2020a
- yalmip version 21-Nov-2017
